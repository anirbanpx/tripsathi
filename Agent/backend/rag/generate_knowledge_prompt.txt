## RULES — read carefully before generating

1. **All prices in ₹ (Indian Rupees).** Use realistic 2024–25 ranges.
2. **Toddler/elderly flags are mandatory** for every attraction. State explicitly if suitable or not, and why.
3. **Trust issues section is mandatory** for every sub-region. Every Indian tourist destination has at least one common pricing trap or booking scam — name it specifically.
4. **Food section must name dishes** — never write "kid-friendly options available" without naming the actual dish.
5. **Driving times must use real highway names or route descriptions** — not just "a few hours."
6. **Best time section must be month-by-month** with concrete implications (monsoon closures, seasonal restrictions, festival crowds).
7. **No generic filler.** Every sentence must be actionable. Do not write "Destination X is a beautiful hill station known for its scenic views."
8. **File length:** aim for 600–900 words. No shorter (too thin for RAG retrieval), no longer (padding dilutes retrieval quality).
